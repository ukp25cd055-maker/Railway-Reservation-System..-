# Automated Railway Reservation System

This is a simple Python based Railway Reservation System developed for UKF College Activity Challenge.

## Features
- Check seat availability
- Book ticket
- View reservation details
- Cancel ticket

## Technologies Used
- Python
- GitHub Version Control

## How to Run
1. Open main.py
2. Run the program in Python
3. Select menu options
